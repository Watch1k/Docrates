КОМПЛЕКСНАЯ ПОМОЩЬ
Институт онкологии EMC – первая частная онкологическая клиника, оказывающая весь комплекс помощи пациентам с онкологическими заболеваниями: диагностика, химиотерапия, хирургия любой сложности, лучевая терапия, реабилитация, психологическая поддержка пациента и его семьи. ЗАРУБЕЖНЫЕ ПРОТОКОЛЫ
Институт онкологии EMC – это лечение рака в Москве по международным стандартам. Лечение проводится по протоколам Национальной Онкологической Сети США (NCCN) и Европейского общества онкологов (ESMO). Для каждого пациента разрабатывается индивидуальный план лечения, согласованный несколькими специалистами.
НОВЕЙШЕЕ ОБОРУДОВАНИЕ
Оборудование последнего поколения для диагностики (ПЭТ/КТ, ОФЭКТ/КТ), лучевой терапии, радиохирургии (TrueBeam), роботизированная хирургия (DA Vinci Si HD).
КВАЛИФИКАЦИЯ СПЕЦИАЛИСТОВ
Ведущие специалисты в области онкологии, радиологии, лучевой терапии, онкохирургии, психоонкологии из США, Европы, Израиля и России.
